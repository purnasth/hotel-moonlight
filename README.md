# Hotel Moonlight

Hotel Moonlight is a beautiful hotel property at Paknajol, Thamel, Kathmandu. It is a 3-star hotel with a beautiful view of the city. The hotel has a total of 20 rooms with a restaurant and a bar. The hotel is located at a prime location with easy access to the city's major attractions. The hotel is known for its hospitality and excellent service.

---

## 🚀 Features

- Dynamic form validation with Yup and React Hook Form
- Google reCAPTCHA integration
- Toast notifications for user feedback
- Fully responsive UI styled with TailwindCSS
- Modular and reusable components
- Optimized for scalability and maintainability

---

## 🛠️ Technologies Used

- **React**: Frontend library
- **Vite**: Fast build tool
- **TypeScript**: For static typing
- **TailwindCSS**: Utility-first CSS framework
- **Yup**: Validation schema
- **React Hook Form**: For form management
- **react-toastify**: For user notifications
- **react-google-recaptcha**: For bot protection

## ⚙️ Setup and Installation

1. Clone the repository:
   ```bash
   git clone 
    ```

2. Install dependencies:
    ```bash
    pnpm install
    ```
3. Create an `.env` file: 
    ```bash
    VITE_API_URL=https://api.com # Your API URL
    VITE_RECAPTCHA_SITE_KEY=6LeIxAcTAAAAAJcZVRqyHh71UMIEGNQ_MXjiZKhI # Your reCAPTCHA site key
    ```
4. Start the development server:
    ```bash
    pnpm run dev
    ```
